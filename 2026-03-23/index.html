领导速览

如果只看 3 分钟，先看这一页。这里把当前最值得优先关注的逻辑按“景气度、确定性、修复弹性”做了拆分。

景气最强

老铺黄金

25 年报只是验证预告，真正超预期的是 26Q1 收入和利润再度大幅抬升。当前主线是品牌高端化兑现后，市场开始重新定价长期空间。

稳健高增

乐舒适 / 亚朵 / 布鲁可

三家公司都属于主业清晰、第二曲线已开始兑现的类型。乐舒适看非洲基本盘+女性护理/拉美，亚朵看酒店+零售双轮，布鲁可看出海+16+与积木车。

修复与重估

巨子生物 / 锅圈

巨子生物更偏“品牌修复+医美新曲线”，锅圈更偏“利润率释放+平台化延展”。两者都已从单一旧框架切换到新估值叙事，但验证节奏仍关键。

老铺黄金

高景气 / 高兑现

25 年报符合预期，但真正的重点在 26Q1 收入和利润再度大超预期。当前市场核心不再是“能不能涨”，而是高景气持续多久，以及高金价回落后动销韧性如何。

26-27 年预期与观点

  • 预计 2026E/2027E 收入分别为 421.5 / 535.1 亿元，同比分别增长 54.4% / 27.0%；净利润分别为 89.0 / 112.5 亿元，同比分别增长 82.9% / 26.4%。
  • 按 2026-03-20 收盘价 611 港元计，对应人民币市值约 949.4 亿元，26E/27E PE 约 10.7x / 8.4x。
  • 评级观点维持中性偏积极，更适合按“回调布局”而非追高交易来理解。

风险点

  • 金价大幅波动。
  • 核心门店高基数压力及新店爬坡不及预期。
  • 高端消费景气度波动。

乐舒适

非洲快消 / 第二曲线

25 年年报超预期，主业不仅稳，而且后八个月还在提速。非洲婴儿护理仍是基本盘，女性护理和拉美市场已经从“讲逻辑”走向“出结果”。

26-27 年预期与观点

  • 预计 2026E/2027E 收入分别约为 6.45 / 7.50 亿美元，同比分别增长 13.8% / 16.3%；净利润分别约为 1.32 / 1.54 亿美元，同比分别增长 9.1% / 16.7%。

风险点

  • 汇率波动与原材料价格上涨。
  • 新市场爬坡不及预期及地缘政治扰动。
  • 份额优先策略下利润率波动加大。

25 年收入基本符合预期、利润略超预期。主线已经从“单一 IP 驱动”切向“多 IP、多年龄层、多区域”共同驱动，第二成长曲线越来越清晰。

  • 预计 2026E/2027E 收入分别约为 38.5 / 47.5 亿元，同比分别增长 32.2% / 23.4%；经调整净利润分别约为 8.5 / 10.7 亿元，同比分别增长 25.9% / 25.9%。

  • 海外扩张不及预期。

锅圈

社区餐饮零售 / 平台化

25 年利润高增兑现，市场开始把锅圈从“火锅烧烤食材加盟店”切换成“社区央厨+多场景家庭餐饮解决方案平台”来定价。

26-27 年预期与观点

  • 预计 2026E/2027E 收入分别约为 98.0 / 116.0 亿元，同比分别增长 25.5% / 18.4%；净利润分别约为 6.2 / 7.7 亿元，同比分别增长 36.6% / 24.2%。
  • 按最新可核验收盘价，对应人民币市值约 89.1 亿元，26E/27E PE 约 14.4x / 11.6x。

风险点

  • 开店不及预期与加盟商盈利能力波动。
  • 消费景气度下行。
  • 其他渠道占比提升对毛利率的压制，以及食品安全和供应链管理风险。

巨子生物

修复反转 / 医美新曲线

25 年进入增长放缓期，但更像高盈利公司做渠道和品牌校正，而不是盈利模式失效。2026 年主线是主业修复与医美商业化能否同步落地。

26-27 年预期与观点

  • 按 2026-03-23 收盘价 29.72 港元计，对应人民币市值约 256.8 亿元，26E/27E PE 约 13.1x / 11.8x。

风险点

  • 新品不及预期与医美商业化推进偏慢。
  • 品牌修复进度低于预期及行业价格竞争加剧。
  • 原料和投放成本上行。

亚朵

酒店 + 零售双轮

25 年收入和利润继续高增，Q4 利润率略超市场预期。零售收入占比已接近四成，亚朵正从单纯酒店公司升级为“住宿 + 生活方式零售”平台。

  • 按 2026-03-20 收盘价 36.88 美元计，对应人民币市值约 370.0 亿元，26E/27E 经调整 PE 约 17.1x / 14.3x。

风险点

  • 加盟商开店意愿回落与商旅需求修复偏弱。
  • 零售竞争加剧。
  • 组织拆分带来的费用压力，以及 RevPAR 修复节奏低于预期。
