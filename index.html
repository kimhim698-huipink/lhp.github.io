老铺黄金

高景气 / 高兑现

25 年报符合预期，但真正的重点在 26Q1 收入和利润再度大超预期。当前市场核心不再是能不能涨，而是高景气持续多久，以及高金价回落后动销韧性如何。

1. 25年报符合预期，26Q1收入和利润均大超预期

2025年公司实现收入273.03亿元，同比增长221.0%；净利润48.68亿元，同比增长230.5%；经调整净利润50.29亿元，同比增长234.9%，整体基本落在此前正面盈利预告给出的收入270-280亿元、净利润48-49亿元区间内，属于符合预期的正式落地。拆分节奏看，2025H2公司实现收入149.49亿元，同比增长199.9%；净利润26.00亿元，同比增长193.7%，高增长在下半年并未明显失速。相比之下，2026Q1更新口径明显更强，公司预计收入165-175亿元、净利润36-38亿元。若以25Q1收入基数68.5亿元测算，26Q1收入同比约140.9%-155.5%；若假设25Q1净利率为18%，则25Q1净利润基数约12.33亿元，对应26Q1净利润同比约192.0%-208.3%。因此，2025年报本身更像是“验证预告”，而26Q1则是实质意义上的大超预期。

2. 25年区域、会员、商场门店和同店表现继续强化，品牌高端化仍在兑现

从区域和渠道结构看，2025年中国内地全渠道收入233.61亿元，同比增长205.4%；海外收入39.42亿元，同比增长361.0%；线下渠道收入226.46亿元，同比增长204.0%；线上渠道收入46.57亿元，同比增长341.3%，线上已经不再只是补充渠道，而是破圈和增量的重要来源。门店端看，截至2025年末公司共有45家门店，分布于16个城市、34个商场，全年新增门店10家、优化及扩容门店9家，其中SKP系门店6家、万象城系门店12家；同店收入同比增长160.6%，单商场年化销售业绩接近10亿元，头部商圈店效继续抬升。会员端同样亮眼，忠诚会员达到61万名，同比增长74.3%，消费人群继续扩容。结合 AlphaPai 召回的卖方跟踪材料，公司消费者与LV、爱马仕、卡地亚、宝格丽等国际五大奢侈品牌的消费者平均重合率已由2025年7月的77.3%提升至2026年3月的82.4%，这意味着老铺黄金的高端定位并非停留在品牌叙事层面，而是在用户结构、商圈层级和复购人群上持续兑现。

3. 26Q1超预期的可能原因，不只是提价和排队，而是发货确认、抢购需求、门店结构和利润率修复的共振

从可能性上看，26Q1超预期首先来自两条核心线索。其一，电商渠道2月底部分发货节奏偏后，收入确认有一部分递延至3月，导致一季度尤其3月确认偏强；其二，2月底提价前后金价快速上冲，抢购行为显著强于原先预期，提价前锁单和排队需求集中释放。结合 AlphaPai 召回的3月纪要与点评材料，可能还存在几条共振因素：一是SKP、万象城等体系内老店延续高增长，老店并未在高基数下明显降温；二是2025H2新增和优化门店主要在上海、香港、新加坡等区域落地，这些门店在26Q1开始完整贡献增量；三是线上渠道延续高增长，且线上客群继续破圈，对收入和利润均形成放大；四是公司在2025年10月第三次提价后毛利率已回到40%以上，2026年2月28日再次提价约25%，叠加镶嵌类和高毛利产品占比提升，使得26Q1利润弹性大于收入弹性。换句话说，26Q1超预期不是单一事件驱动，而更像品牌势能、渠道扩容、价格传导和产品结构同时发力的结果。

4. 26-27年预期与估值

按照桌面模型的中性偏乐观情形，预计公司2026E/2027E收入分别为421.5亿元和535.1亿元，同比分别增长54.4%和27.0%；净利润分别为89.0亿元和112.5亿元，同比分别增长82.9%和26.4%。按最新可核验收盘价，2026年3月20日公司股价为611港元，对应人民币市值约949.4亿元，对应2026E/2027E PE约10.7x/8.4x。我们维持中性偏积极判断，更适合按“回调布局”而非追高交易来理解；若后续动销和客流在金价回落后仍具韧性，当前估值仍有吸引力。风险点主要包括金价大幅波动、核心门店高基数压力、新店爬坡不及预期以及高端消费景气度波动。

1. 25年收入、毛利和净利润均超市场预期，后八个月增速进一步提速

2025年公司实现收入5.67亿美元，同比增长24.9%；毛利2.04亿美元，同比增长27.1%；净利润1.21亿美元，同比增长27.4%；经调整净利润1.22亿美元，同比增长24.4%；毛利率35.9%，同比提升约0.6-0.7pct；净利率21.4%，同比提升0.4pct。结合 AlphaPai 召回的卖方点评，此前市场对公司2025年收入增速普遍按接近20%预期，因此正式年报属于超预期落地。拆分节奏看，公司2025年前四个月收入1.61亿美元、净利润0.31亿美元，对应后八个月收入4.06亿美元，同比增长29.0%，净利润0.90亿美元，同比增长33.5%，说明全年增长并非前高后低，而是下半年继续提速。

2. 主业仍由婴儿护理驱动，但卫生巾和湿巾的增速更快，第二成长曲线已经开始兑现

从品类看，2025年婴儿护理收入4.46亿美元，同比增长23.1%，占比78.6%，仍是绝对基本盘；女性护理收入0.99亿美元，同比增长27.9%，占比17.5%，较2022年的13.5%继续抬升；家庭护理收入0.22亿美元，同比增长53.8%，占比3.9%，体量虽小但弹性最强。销量端同样验证这一趋势，婴儿护理、女性护理和家庭护理销量分别增长17.9%、19.4%和52.8%。其中，女性护理不只是自然增长，2025年公司积极参与加纳政府免费卫生巾派送项目，而加纳政府预计2026年至2029年投入超过8亿加纳塞地采购卫生巾，这意味着卫生巾业务后续不仅有渗透率提升逻辑，还有项目放量和品牌协同逻辑。

3. 区域扩张与渠道深化同时兑现，非洲稳增长、拉美高弹性、中非仍是更快一档的区域

区域上，公司2025年东非、西非、中非和拉美收入分别为2.56亿、2.31亿、0.58亿和0.22亿美元，同比分别增长23.9%、18.4%、34.5%和134.0%。非洲仍是基本盘，但结构上已经不是单纯靠成熟市场硬推增长，而是东非、西非稳健增长，中非更快，拉美开始进入放量期。以秘鲁为例，2025年销售额从2024年的930万美元增长至1730万美元，同比增长85.8%，公司还在当地加快商超和药店等现代渠道布局。与此同时，公司已开始把CRM系统从渠道商延伸至终端门店，并率先在加纳和肯尼亚试点，这说明乐舒适的增长模式正在从“铺渠道”走向“渠道数字化+终端精细化运营”，后续对产品组合和动销效率的提升更值得跟踪。

4. 26-27年预期与估值

按 AlphaPai 召回的近期卖方预测中枢看，预计公司2026E/2027E收入分别约为6.45亿美元和7.50亿美元，同比分别增长13.8%和16.3%；净利润分别约为1.32亿美元和1.54亿美元，同比分别增长9.1%和16.7%。按最新可核验收盘价，2026年3月13日公司股价为28.46港元，对应人民币市值约163.7亿元，对应2026E/2027E PE约17.2x/14.7x。我们维持中性偏积极判断，核心看点仍是非洲基本盘稳固下，女性护理、拉美和终端精细化运营能否继续打开第二成长曲线。风险点主要包括汇率波动、原材料价格上涨、新市场爬坡不及预期、地缘政治扰动，以及份额优先策略下利润率波动加大。

乐舒适公司交流核心要点

  • 管理层将公司定位从“非洲卫生用品龙头”进一步上修为“非洲领先快消品集团”，强调核心壁垒是本地制造、本地供销体系，以及对当地政治、经济、渠道和消费习惯的深度理解。
  • 未来一段时间收入、毛利和净利润仍以 mid-teens 增长为基线，中非增速快于东非，拉美是未来两三年最重要的增量观察点。
  • 公司明确坚持“份额优先”，若短期利润与长期份额冲突，会优先做长期份额的平衡。
  • 短期仍以 Softcare 为核心主品牌，但长期会走向多品牌矩阵；SKU增加很大程度来自不同国家法规、标签和渠道适配要求。
  • 安全库存加海上在途大致有四个多月，新工厂在已有渠道基础上投产后有机会较快爬至70%-80%产能利用率。
  • 在没有重大并购的情况下，分红率倾向维持在净利润45%左右，同时继续推进非洲和拉美产能布局。
  • 管理层认为未来三年难有新进入者在多个国家同时形成系统性冲击，反映其对本地化壁垒和渠道深度有较强信心。

布鲁可

潮玩积木 / 出海扩张

25 年收入基本符合预期、利润略超预期。主线已经从单一 IP 驱动切向多 IP、多年龄层、多区域共同驱动，第二成长曲线越来越清晰。

1. 25年收入基本符合、利润略超卖方预期，25H2新品加速后经营趋势继续向上

2025年公司实现收入29.13亿元，同比增长30.0%；毛利13.64亿元，同比增长15.7%；年内利润6.34亿元，较2024年的亏损3.98亿元明显扭亏；经调整净利润6.75亿元，同比增长15.5%；毛利率46.8%，同比下降5.8pct；经调整净利率23.2%，同比下降2.9pct。结合 AlphaPai 召回的卖方点评，市场对公司2025年收入大体按“符合预期”看待，但利润端普遍按“略超预期”解读，核心原因是25H2推新速度明显加快，虽然毛利率承压，但新品研发逐步出现规模效应，同时销售和管理费用控制好于部分悲观预期。拆分节奏看，25H2公司实现收入15.75亿元，同比增长31.9%；经调整净利润3.55亿元，同比增长21.3%，无论收入还是利润都比上半年更快，说明2025年不是单纯的存量兑现，而是新品、出海和高年龄层产品在下半年共同发力。

2. “全人群+全价位”策略已经跑通，但平价带放量和密集推新也阶段性压制了利润率

从产品结构看，2025年拼搭角色类玩具收入28.43亿元，同比增长29.1%，仍是绝对主体；拼搭载具类玩具及积木玩具收入0.70亿元，同比增长77.0%，其中积木车于2025年11月上市后仅两个月就贡献4310万元收入，验证了新品类放量能力。年龄层上，6-16岁产品仍贡献81.1%收入，但16岁以上产品收入占比已由2024年的11.4%提升至2025年的16.7%，成人向产品成为清晰的第二增长曲线。价格带上，9.9元平价产品2025年收入达到5.41亿元，占总收入18.6%，销量达到1.22亿件，占总销量47.8%，已经成为最重要的拉新抓手。问题在于，平价带占比提升、新品磨合期模具折旧增加、IP成本摊销加大，使得毛利率和经调整净利率阶段性回落，因此布鲁可当前更像是“收入和用户数先冲、利润率后修复”的状态。

3. 出海、IP多元化和渠道结构优化同时推进，业务韧性明显强于单一IP驱动阶段

区域上，公司2025年国内收入25.94亿元，同比增长19.2%；海外收入3.19亿元，同比增长396.6%，海外收入占比从2024年的2.9%提升至2025年的10.9%。其中亚洲（不含中国）收入1.33亿元，同比增长238.1%；美洲收入1.50亿元，同比增长804.1%，美国和印度尼西亚已成为最重要的两个海外市场。渠道上，线下经销收入26.37亿元，占比90.5%，同比增长27.8%；线上收入2.53亿元，同比增长62.3%，增速显著快于整体，也更适合16+产品销售。IP结构上，截至2025年末公司拥有73个授权IP、29个已商业化IP，前四大IP收入占比从2024年的91%-92%下降至2025年的81%-83%，其中变形金刚收入9.51亿元，同比增长约110%，假面骑士收入3.31亿元，同比增长约95%，奥特曼收入8.15亿元，同比下滑约26%，英雄无限收入2.64亿元，同比下滑约15%。这说明布鲁可已经从“高度依赖少数爆款IP”的阶段，开始切向“多IP、多年龄层、多区域”共同驱动的新阶段。

4. 26-27年预期与估值

按 AlphaPai 召回的华泰、中信建投及外资点评中枢看，预计公司2026E/2027E收入分别约为119亿元和142.5亿元，同比分别增长21.6%和19.7%；经调整净利润分别约为21.6亿元和25.9亿元，同比分别增长23.2%和19.9%。按最新可核验收盘价，2026年3月20日公司股价为36.88美元，对应人民币市值约370.0亿元，对应2026E/2027E经调整PE约17.1x/14.3x。我们维持中性偏积极判断，后续仍重点看酒店扩张、零售延展和RevPAR修复三条主线能否继续共振。风险点主要包括加盟商开店意愿回落、商旅需求修复偏弱、零售竞争加剧、组织拆分带来的费用压力，以及RevPAR修复节奏低于预期。
